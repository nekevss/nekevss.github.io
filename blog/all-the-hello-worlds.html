An Abundance of Hello World's

8/10/2023

Let's write some hello world's in a bunch of languages that I might be posting on. Why? Well I definitely am not using this blog to test pre tag handlings and code themeing at all ... nothing to see here.

Rust


fn main() {
    println!("Hello World!");
}

JavaScript


// Yes, this is a bit much. But console.log("Hello World!") is boring.
(function() {
    console.log("Hello World!")
})()

Go


func main() {
    fmt.Println("Hello World!")
}

Python


# Meh, it's just python
print("Hello World!")

C


include <stdio.h>

int main() {
    printf("Hello World!\n")
    return 0
}

Zig


// To be honest, still not sure whether `std` or `print` import is best.
const print = @import("std").debug.print;

pub fn main() void {
    print("Hello World!" .{});
}
